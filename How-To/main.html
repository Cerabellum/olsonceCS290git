Introduction

Web Security is an important element of web development and programming. It is an element of most sites due to the increased desire to keep information private. Encryption and decryption are crucial to keeping data secured. CryptoJS, a Javascript encryption and decryption library, covers all the well known algorithms, including SHA, DES, 3DES, AES, and several others. It is still a developing library, but is one of the libraries that is still being activiely worked on and shows a lot of promise.

We'll discuss implementing the following Algorithms:

  • GetNewsForApp
  • GetPlayerSummaries
  • GetFriendList
  • GetPlayerAchievements
  • GetUserStatsForGame
  • GetOwnedGames
  • GetRecentlyPlayedGames

While the library covers many other algorithms, these are the most popular ones and gives a good example of the implementation of this library.
