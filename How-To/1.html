Welcome to a How-To for using Steam's Web API

by Andrew Rice

About this Guide

or "How to use the How-To"

Welcome to my How-To guide for Steam's Web API.

Steam's Web API is a way to access data about steam users. Examples of the data you can access are things like lists of owned games, achievements, profile information, etc. This is all done through either JSON or XML objects returned from various queries. This guide's aim is to take a tour of all the different information you can access and various examples of what you can do with it.

Code examples will mostly use PHP. The more important part is seeing what kind of information you have to send to receive what you need in return, however, and thus it is important to keep in mind you are not limited to using these languages. As such, we will be using the simplest of the simple: PHP's file_get_contents() function to return a JSON object. Typically, you won't want to do this since it will freeze the rest of the page load while it gets the return from the API, but it serves our purposes here.

We'll discuss the following JSON calls:

  • GetNewsForApp
  • GetPlayerSummaries
  • GetFriendList
  • GetPlayerAchievements
  • GetUserStatsForGame
  • GetOwnedGames
  • GetRecentlyPlayedGames

We'll leave out GetSchemaForGame, IsPlayingSharedGame, GetGlobalStatsForGame, GetGlobalAchievementsPercentagesForApp. This is primarily because the information returned isn't particularly interesting or useful. If you happen to be interested, you can find those call syntaxes here.
